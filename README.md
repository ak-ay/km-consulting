# K M Consulting Website

Glassmorphic static website built from the supplied 2026 content PDF.

## Files

- `index.html` - page structure and content
- `styles.css` - editorial SaaS visual system with off-white grid, black CTAs, terracotta accents, and motion-ready sections
- `script.js` - mobile navigation, reveal behavior, active nav state, and quote email handoff

## Preview

Open `index.html` in a browser, or run:

```bash
python3 -m http.server 4173 --bind 127.0.0.1
```

Then visit `http://127.0.0.1:4173/`.
